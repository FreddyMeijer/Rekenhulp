#### Keuze tafels tot
Bij de keuze *tafels tot* geef je aan wat de maximale vermenigvuldiger is en wat het maximale vermenigvuldigtal is. Dus als je hier kiest voor 15, is de hoogst mogelijke vermenigvuldiging `15*15`.
